# “吃点好的” — 智能治愈系饮食记录应用设计文档 (v2.1 Spotlight 优化版)

本项目是一款面向日常生活记录的智能饮食 Web 应用，以“吃点好的”为核心主题。将日常饮食记录转化为“可回忆、可收藏的生活回忆录”。

---

## 1. 核心视觉与页面规划

### 1.1 视觉风格 (UI/UX)
* **配色调性**：采用柔和温润的粉彩马卡龙色调。
  * 背景色：暖奶油白（#FFFDF6）辅以可爱的网格线背景，模拟手账本的网格纸纹理。
  * 卡片及辅助色：粉色（#FFB6C1）、薄荷绿（#A8E6CF）、香蕉黄（#FFD3B6）。
  * 交互体验：大圆角卡片（`border-radius: 24px`），按钮和过渡带有弹性 bouncy 动画。
* **Logo 设计**：
  * 顶部 Logo 采用手绘圆体排版，左侧配有由纯 CSS 和 SVG 绘制的**热气腾腾小面碗/可爱笑脸饭团**图形，文字下方带有波浪虚线装饰。
* **可爱字体引入**：
  * 全局引入 Google Fonts 的中文字体 **"ZCOOL KuaiLe" (站酷快乐体)**，英文及数字搭配 **"Fredoka"**，使整个界面具有萌系治愈的手账笔触风格。
* **Spotlight 压暗背景食物高亮效果**：
  * **不再使用透明抠图贴纸**。为了保留照片真实丰富的背景和生活感，我们保留完整的原始照片，但通过 Canvas 将背景区域压暗（叠加一层柔和的深色遮罩），而将食物主体保持明亮高光，从而优雅地突出食物。
  * **不规则边缘描边**：在明暗交界处（即食物主体边缘），利用 Canvas 提取食物轮廓，绘制 4px 白色虚线描边。由于去除了画布边缘的噪声干扰，该虚线将完美贴合食物的物理轮廓，呈现精美的手账手绘勾画质感。

### 1.2 四大核心功能板块
1. **【食光时间轴 - Today】(首页)**
   * **结绳记事概念**：采用一条连贯的垂直时间轴。每一个日期是一个圆形的“绳结节点”（例如“7月2日”）。
   * 日期节点下方依次展开该天所有的饮食卡片。
   * 用户可以直接向下滑动浏览过去的所有历史记录，彻底取消顶部的时间筛选翻页器。
2. **【月视图】**
   * 以传统的日历网格展示整月。
   * 每个日期格中会平铺展示当天所有饮食记录的食物贴纸缩略图，形成一面当月食物贴纸墙。
   * 点击日期格直接平滑滚动定位到首页该日期的位置。
3. **【回忆录】(月报模式)**
   * 自动生成一份精美的长图卡片，展示当月的数据回忆，并调用 AI（Gemini）根据用户当月的饮食历史、聚餐频次和心情，生成一段温暖治愈的月度总结。
4. **【收藏推荐页】**
   * 展示所有被用户标记为“收藏”的饮食回忆卡片。

## 2. 核心交互流程

### 2.1 Spotlight 食物高亮与边缘描边算法
1. 用户上传图片，我们同时持有**原始图片**和**AI去背景抠图（仅食物部分）**。
2. 将原始图片绘制在 Canvas 底层。
3. 覆盖一层半透明深色遮罩（如 `rgba(0, 0, 0, 0.4)`）。
4. 在最上层绘制 AI去背景抠图（位置与底图完全对齐）。此时，背景被遮罩压暗，而食物主体保持明亮，达成 Spotlight 聚光灯高亮效果。
5. **边缘提取算法优化**：
   - 为防止追踪器贴到画布边缘绘制矩形框，在检测像素透明度时，强制画布外圈 5px 宽度为透明。
   - 使用 Moore-Neighbor 边缘追踪算法提取食物本身的不规则边界点集。
   - 对点集进行双重均值平滑处理，并在法线方向向外偏移 8px，使用 `ctx.setLineDash([8, 6])` 绘制白色虚线，完美包裹在亮色食物周围。
6. **兜底高亮（Fallback）**：若 AI 去背景模型加载失败或超时，在照片中央使用圆形渐变遮罩（径向渐变，中心透明、四周暗色）作为聚光灯，并在中心区域绘制圆形虚线框。
```
底图 (原图)  ---> 压暗层 (rgba) ---> 顶图 (去背景食物) ---> 轮廓描边 (Canvas 边缘虚线)
```
