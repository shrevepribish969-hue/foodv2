# 抠图纯贴纸化与自适应紧贴描边设计

本设计旨在解决手账风食物记录应用中，食物抠图后的描边在浅色背景上不明显的问题，并对抠图效果进行手账贴纸化与描边精准化改良：
- **纯贴纸化**：抠图成功时，完全丢弃原始大图的背景（使背景变透明），只保留抠出来的食物本体。
- **自适应紧贴描边**：使用 **偏移剪影描边法 (Offset Silhouette Outlining)** 在食物边缘生成一圈 100% 精准、紧贴食物边界、宽度为 3px 的实线，颜色使用食物像素平均色乘以 `0.4` 加深后的颜色。

## 方案设计

### 1. 纯贴纸化与颜色计算

修改 [RecordModal.tsx](file:///d:/A研二/food2.0纯记录/foodV2/src/components/RecordModal.tsx) 的 `renderCanvas` 函数。
原本的绘制逻辑为：先绘制底层原图 -> 叠加压暗遮罩 -> 绘制抠图食物 -> 绘制不精确的追踪虚线。

我们将绘制逻辑修改为：
- **抠图成功时（`imgCutout` 存在）**：
  1. 不绘制底层原图 `imgOriginal`，不绘制压暗遮罩。
  2. 提取不透明像素并计算平均色：遍历 `tempData`（非背景透明像素），统计透明度 `alpha > 40` 的像素。累加其 R, G, B 值，计算平均 RGB 分量，并各乘以 `0.4` 得到加深描边色 `strokeColor`。
  3. 创建离屏 Canvas（`silhouetteCanvas`），绘制 `imgCutout`，并将 `globalCompositeOperation` 设置为 `'source-in'`，填充 `strokeColor`。得到一张食物的纯色剪影。
  4. 在主 Canvas 上，将 `silhouetteCanvas` 朝 8 个方向（上、下、左、右、左上、右上、左下、右下）各偏移 3px 进行绘制，合成一圈 3px 宽的纯色边缘底图层。
  5. 在正中心（零偏移）绘制彩色食物 `imgCutout` 覆盖其上，从而在边缘露出 3px 的完美加深色描边线。
- **抠图失败兜底时（`imgCutout` 不存在）**：
  1. 绘制底层原图 `imgOriginal`。
  2. 叠加径向渐变聚光灯（中心亮、四周暗）。
  3. 绘制圆形手绘风虚线圈，描边颜色改为温暖的手账深褐色 `#5C4B43`。

### 2. 废弃冗余算法

删除原本在 `RecordModal.tsx` 中定义的连通域边缘追踪函数 `traceContour` 与均值平滑函数 `smoothPoints`，使文件结构更加精炼。

## 验证计划

- 上传一张食物图片，观察抠图完成后的预览图是否为透明背景的食物贴纸（没有原图背景），且在食物边缘有一圈 3px 宽、贴合度完美的加深色实线描边。
- 测试 WASM 抠图失败时的兜底（圆形虚线）描边，确保原图可见且描边为温暖的深褐色 `#5C4B43`。
