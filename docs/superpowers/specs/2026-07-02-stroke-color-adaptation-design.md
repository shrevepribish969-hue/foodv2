# 手账贴纸双层白边与明丽橘黄色描边设计 (抗糊去噪升级版)

本设计旨在根据用户的参考图，实现兼具可爱感与手账质感的“双重贴纸描边”效果，并精确处理盘子与背景桌面的边缘：
- **边缘去噪硬化**：在生成任何描边图层前，使用 **Alpha 阈值过滤算法 (Alpha Thresholding)** 处理抠图。将不透明度 `Alpha < 40` 的淡色桌面背景残留清零，将 `Alpha >= 40` 的餐具（如白色纸盘子、碗）与食物区域的像素强行设为完全不透明 (`255`)。这能确保即使餐具被轻微抠淡，其边缘也能被完整保留并进行描边，从根本上解决“盘子在画面里，但描边只缩进到蛋糕上”的尴尬断层。
- **白色扩边底**：在去噪处理后的餐具与食物最外侧边缘外扩展一圈 8px 宽的纯白色底框。
- **橘黄色边缘线**：在白色边框的最外侧，描画一圈厚度为 3px 的明丽橘黄色（`#FF9800`）连续实线。
- **背景透明化**：抠图成功时保持背景透明，使最终呈现为一张带白边和橘边的高品质食物贴纸。

## 方案设计

### 1. 双层描边渲染流程

修改 [RecordModal.tsx](file:///d:/A研二/food2.0纯记录/foodV2/src/components/RecordModal.tsx) 的 `renderCanvas` 函数。
在 `imgCutout` 存在时：
1. 不绘制底层原图 `imgOriginal`，不绘制压暗遮罩。
2. **像素级阈值去噪**：创建 `cleanCanvas`，绘制 `imgCutout`。利用 `getImageData` 读取并遍历像素，若透明度 `Alpha < 40` 直接强行设为 `0`；若 `Alpha >= 40` 强行设为 `255`。通过 `putImageData` 将干净且边缘锋利的剪影数据刷回 `cleanCanvas`。
3. **绘制橘黄色剪影**：创建 `orangeSilhouette` 离屏 Canvas，绘制 `cleanCanvas` 并使用 `'source-in'` 填充明丽的橘黄色（`#FF9800`）。
4. **绘制白色剪影**：创建 `whiteSilhouette` 离屏 Canvas，绘制 `cleanCanvas` 并使用 `'source-in'` 填充纯白色（`#FFFFFF`）。
5. 在主 Canvas 上朝 16 个方向平移 `11px`（8px 白框 + 3px 橘线）绘制橘黄色剪影。
6. 在主 Canvas 上朝 16 个方向平移 `8px` 绘制白色剪影，盖在橘黄色图层上方。
7. 最后在正中心（0px 偏移）绘制彩色食物 `imgCutout`。

抠图失败兜底时（`imgCutout` 不存在）：
- 显示原图并叠加径向压暗，并用橘黄色（`#FF9800`）绘制圆形虚线描边。
